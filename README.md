# أسهَر 🦅

**عينك اللي ما تنام — يصيد ساهر قبل لا يصيدك.**

تطبيق ويب (PWA) عربي ينبّهك صوتياً قبل كاميرات ساهر في السعودية، مبني خصيصاً لخط جدة–ينبع.

## 📱 الرابط الرسمي (للعائلة)

**https://non-rtr.github.io/ashar/**

### تثبيته على الآيفون (مرة وحدة)
1. افتح الرابط بـ **Safari**
2. زر المشاركة ⬆️ ← **«إضافة إلى الصفحة الرئيسية»**
3. افتحه من أيقونة الصفحة الرئيسية، اسمح بالموقع، وارفع كتم الصوت 🔊

### نظام العائلة
- أول ما تعلّم كاميرا بزر «كاميرا هنا» يسألك اسمك (مرة وحدة)
- من الإعدادات: **«شارك كاميراتك مع العائلة»** ← يطلع رابط أرسله بقروب الواتساب
- أي واحد يفتح الرابط تنضاف الكاميرات لجهازه تلقائياً باسم اللي علّمها
- بيانات العائلة ما تطلع لأي سيرفر — أجهزتكم وروابطكم فقط

## التشغيل محلياً

```bash
cd ~/Projects/ashar
python3 -m http.server 8089
# افتح http://localhost:8089
```

> الموقع الجغرافي يشتغل فقط على `localhost` أو HTTPS — لاستخدامه على الجوال في السيارة يحتاج استضافة HTTPS (اطلبها وقت ما تبي).

## وضع التجربة

زر «وضع التجربة (جدة)» يحاكي قيادة بسرعة 120 كم/س عبر كاميرات جدة عشان تشوف وتسمع التنبيهات وأنت بمكانك. للتطوير: `?demo=1&fast=8`.

## مصدر البيانات — كن صريحاً مع نفسك

- **289 كاميرا ثابتة** من OpenStreetMap (ODbL) — أغلبها داخل المدن.
- **خط جدة–ينبع السريع: صفر كاميرات موثّقة في OSM.** ما اخترعنا إحداثيات وهمية.
- الحل: زر **«كاميرا هنا»** — ضغطة وحدة وقت ما تمر على كاميرا تحفظها للأبد (تنحفظ محلياً وتُصدَّر/تُستورَد JSON، ويدعم استيراد GPX/CSV من أي مصدر خارجي تشتريه).
- الكاميرات المتنقلة ما يمكن رصدها بقاعدة ثابتة.

## التنبيهات

- مرحلة 1 (~25 ثانية قبل الكاميرا): بيب مزدوج + «ساهر بعد X متر»
- مرحلة 2 (~10 ثواني): بيب ثلاثي حاد + «قريب! الحد Y»
- تجاوز السرعة قرب كاميرا: وميض أحمر + نغمة إنذار + «خفّف السرعة»
- المسافات تتكيّف مع سرعتك (وقت ثابت، مو مسافة ثابتة)
